# wcmlplanner
WEST COAST MAIN LINE

WHY?
As the main intercity rail route in the UK, a multitude of service patterns and companies operate on the line. The problem is that things get so complex, no one can easily grasp handy information. For example, how many trains are there per hour from Milton Keynes to Stoke-on-Trent? What time is best to go from Smethwick to Windermere (it's every 2h)? Also, train company websites aren't very visual: key in A to B, but how far is the trip up the country?
This website aims to create a user-friendly platform for obtaining information on service patterns, journey time, timetables, etc., with key connecting routes.
